# How to run the Website (WebApp/System)

## Please ensure you have installed the following:
- Python 3 : https://www.python.org/downloads/
- NodeJS : https://nodejs.org/en/download/

Verify this by running the following in your terminal/ CMD separately:

    python3 --version

    node --version


## Then in the project's/code's directory run the following command: ( Using terminal / CMD )

    python3 setup.py

## In case you get any errors, and you did as said above, restart your computer and try again.

## Other useful commands:

# run application after setup:
    python3 run_application.py
# run frontend only:
    python3 run_frontend.py
# run backend only:
    python3 run_backend.py

# setup backend:
    python3 setup_backend.py
# setup frontend:
    python3 setup_frontend.py
